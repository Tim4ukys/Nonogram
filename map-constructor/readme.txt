~~~ about ~~~
Инструмент, с помощью которого можно создавать или просматривать карты.

ВАЖНО: Инструмент проверялся только на OC: Arch Linux и Manjaro Linux.
Корректная работа на других устройствах пока не гарантирована.

~~~ Установка зависимостей ~~~
pamac install sfml imgui-sfml

~~~ Создание карты ~~~
Запускаем приложение. Двигая ползунок "Ширина и высота поля" изменяем размер поля (!). Тыкаем на нужный квадрат и он закрашивается.
Чтобы сохранить полученную карту, нужно нажать на ... (на что же бл*ть?) ... на кнопку "Сохранить поле" (!).
Поле сохранится в файл "out_map.txt".

~~~ Загрузка карты ~~~
Достаём любой файл карты и копируем его в папку, в которой лежит инструмент. Переименовываем его в "out_map.txt".
При запуске инструмента он сам загрузит его.
